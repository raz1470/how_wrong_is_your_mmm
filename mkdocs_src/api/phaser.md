# BudgetPhaser and Blackout

`BudgetPhaser` recommends the weekly spend phasing needed to reduce
elasticity uncertainty, while preserving monthly totals exactly.
`Blackout` is one of the deviation shapes it accepts per channel — a hard
on/off switch instead of a continuous weekly range.
